# assets/

- `welcome.jpg` — прикрепляется к приветственному сообщению после `/start`
  (handlers/common.py, WELCOME_PHOTO).
- `pro.jpg` — прикрепляется к сообщению-питчу ⭐ F1 Bot Pro (`/pro`, меню →
  «⭐ F1 Bot Pro»). Расширение может быть `.jpg`, `.jpeg`, `.png` или `.webp`
  (см. render.pro_photo()).

Если файла нет — бот не падает, просто отправляет обычный текст без фото.
Это НЕ фото самого счёта оплаты (кнопка «Заплатить») — там Telegram требует
публичную HTTPS-ссылку, а не файл; сейчас счёт идёт без фото.
