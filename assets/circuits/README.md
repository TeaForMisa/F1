# assets/circuits/

Необязательные локальные схемы трасс для страницы этапа (Меню → Календарь → этап).

- Имя файла = `circuitId` из API + расширение: `<circuit_id>.png` (или `.jpg`, `.jpeg`, `.webp`).
  Примеры: `silverstone.png`, `monaco.png`, `madring.png`.
- Локальный файл имеет **приоритет** над официальной схемой с сайта F1.
- Если локального файла нет — берётся официальная схема F1 по ссылке.
- Если и её нет (например, у нового «Мадринга» — `madring`) — страница показывается
  текстом без изображения.

circuitId можно посмотреть в `circuits.py` (ключи словаря `CIRCUITS`).
